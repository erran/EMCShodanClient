    Welcome to my Objective-C library for Shodan.

    This library was the first major project I built but I had a whopping three APIs to look at and play with to see what functionality I needed. Best of all the author, John Matherly (@achillean), of Shodan was great at responding and giving me help.

    Usage

    Before you can use this Objective-C library to connect to the Shodan API, you need to have an API key from SHODAN.

    Create and initialize an instance of the WebAPI class:

    #import "api.h" // Import the Shodan API
WebAPI* api = [[WebAPI alloc]init_with_api_key:@"Your API Key"];

    Print a list of cisco-ios devices:

    NSDictionary* result = [api search:@"cisco"];
int i = 0;
for (id host in [result objectForKey:@"matches"]) {
    host = [[result objectForKey:@"matches"] objectAtIndex:i];
    i++;
    NSLog(@"%@",[host objectForKey:@"ip"]);
}

    Grab all information Shodan has on a specific ip (4.59.125.121):

    NSDictionary* host = [api host:@"4.59.125.121"];
NSLog(@"%@",host);
